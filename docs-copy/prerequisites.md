# Prerequisites

- PHP >= 8.0
- Laravel >= 9.x
- Composer
- Safaricom M-Pesa API credentials (sandbox or production)
- Ngrok (for local callback testing)
- Postman or similar API testing tool

## Recommended
- Git for version control
- A secure server for production deployments
